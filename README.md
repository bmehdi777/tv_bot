# Skynet

A bot for Telegram written in python.

## Getting started
### Prerequisites
Install pip package :
- Beautifulsoup4
- dryscrape
- xmltodict
- weather-api
- python-telegram-bot

You can install them manually or use the :
>pip install -r requirement

### Installing
You absolutely need to run the "install.sh" file :
>./install.sh

## Built with

- Python
- Telegram-API
