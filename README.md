# Skynet

A bot for Telegram written in python.

## Getting started
### Prerequisites
Install pip package :
- Beautifulsoup4
- lxml
- xmltodict
- xvfbwrapper
- weather-api
- six
- urllib3
- python-telegram-bot
- webkit-server
- idna
- future

### Installing
You absolutely need to run the "install.sh" file.
>./install.sh
